AI-Powered Cybersecurity Solutions

Beryllium Security offers cutting-edge tools that revolutionize how cybersecurity professionals approach offensive security. Our solutions—including the open-source Nebula and advanced Nebula Pro—streamline vulnerability assessments while prioritizing data privacy.

Nebula – Open-Source Penetration Testing Assistant

Key Features

  • Free & Open-Source: BSD-2-Clause licensed for both individuals and organizations.
  • Natural Language Interface: Execute commands using everyday language.
  • Terminal-Embedded AI: Seamlessly integrated within your CLI for efficient operations.
  • Real-Time Insights: Provides context-aware recommendations as you work.
  • Autonomous Mode (Preview): Chains multiple tools for automated tasks.
  • Local AI Processing: Keeps your data private by leveraging local models (e.g., Meta's Llama, Mistral).
  • GitHub Repository: View on GitHub

Current version: Nebula 2.0.0b4 (Feb 4, 2025)

Nebula Pro – Advanced PenTestOps Platform

Features

  • All-in-One Platform: Integrates every phase of penetration testing from reconnaissance to reporting.
  • Folder-Level Engagements: Automatically organizes data for a streamlined workflow.
  • AI-Powered Assistants:
    • Terminal Assistant: Generate and analyze scripts right in your CLI.
    • Suggestion Assistant: Recommends next steps based on real-time outputs.
    • Note Assistant: Summarizes findings and prepares report notes.
  • Autonomous Mode: Automatically charts attack paths with your oversight.
  • Collaboration & Multitasking: Supports parallel engagements with comprehensive audit trails.
  • Integrated Tools:
    • Screenshot & Image Editor for capturing and annotating outputs.
    • Privacy Controls such as Incognito Mode and PII masking.

Interested organizations can request a trial or view our pricing details.

DAP – Deep Application Profiler for Malware Analysis

Features

  • AI-Driven Analysis: Reverse-engineers executables to detect malicious behavior.
  • Dual Offerings: Web interface for analysts and API for enterprise integration.
  • Novel Threat Detection: Identifies zero-day or custom malware beyond traditional signatures.
  • Focused Analysis: Supports executables up to ~0.5 MB with emphasis on unmanaged code.
